# learning-API
API learning

**How To Run**
- Clone this Repo
- Cd /api/
- run in terminal **python api.py**

and will start server on 127.0.0.1:5000

: Endpoint
----------------
1. **Authentications** :
    - User Login
    - User Register
    - User Logout*
    - Forgot Password*
   
2. **User **:
    - Get User Mahasiswa
    - Update User Profile

3. **Class**
    - Get All Class
    - Get Class
    - Get Post Class
    - Post articel in Class
    - Edit articel in Class
    - Delete articel in Class 
    - Join to Class
    - Accept User
    - Comment Post
    - Delete Comment
    - Edit Comment
